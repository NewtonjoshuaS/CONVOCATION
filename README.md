**EASTC CONVOCATION CEREMONY**
This project is a static HTML webpage designed to present information about the EASTC Convocation Ceremony.
It displays the event schedule, speakers, ceremony theme, venue details, and official contact information.

**Project Description**
The webpage provides a simple and clear layout for:
Convocation ceremony schedule
Event details (date, time, venue)
Speakers section with images
Ceremony theme and description
EASTC contact and website link
The project is built using HTML and CSS only and does not require any backend or database.
